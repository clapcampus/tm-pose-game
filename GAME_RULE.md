# 게임 기획: Fruit Catcher (과일 받기 게임)

## 1. 게임 개요
- **제목**: Fruit Catcher (프루트 캐처)
- **장르**: 아케이드 / 포즈 컨트롤
- **목표**: 하늘에서 떨어지는 과일을 바구니로 받아 점수를 획득하고, 폭탄을 피해서 오래 살아남는 것.

## 2. 조작 방법 (Pose)
사용자의 포즈를 인식하여 바구니를 이동시킵니다.
- **왼쪽 기울기/손들기 (Label: `LEFT`)**: 바구니를 왼쪽으로 이동
- **가만히 서기 (Label: `CENTER`)**: 바구니를 중앙으로 이동
- **오른쪽 기울기/손들기 (Label: `RIGHT`)**: 바구니를 오른쪽으로 이동

## 3. 화면 구성 (Split Layout)
화면을 좌우로 나누어 배치합니다.

### A. 왼쪽 패널 (System Zone)
- **웹캠 화면**: 내 모습과 포즈 인식 상태 확인 (Mirror Mode)
- **인식 결과**: 현재 인식된 Label (LEFT / CENTER / RIGHT) 및 확률
- **컨트롤**: 게임 시작/종료 버튼, 설정(옵션)

### B. 오른쪽 패널 (Game Zone)
- **게임 화면**: 실제 게임이 진행되는 공간 (Canvas)
- **3 Lane 구조**: 바구니가 이동하는 3개의 라인
- **상단 정보**: Score, Level, Time
- **중앙**: 떨어지는 과일과 바구니 캐릭터

## 4. 아이템 및 점수 규칙
하늘에서 랜덤하게 아이템이 떨어집니다.

| 아이템 | 설명 | 점수/효과 |
|:---:|:---|:---:|
| 🍎 **사과** | 기본적인 과일 | +100점 |
| 🍊 **오렌지** | 맛있는 과일 | +200점 |
| 🍇 **포도** | 보너스 과일 | +300점 |
| 💣 **폭탄** | 피해야 할 장애물 | **Life -1 (또는 게임오버)** |

## 5. 게임 진행 규칙
- **레벨 디자인**:
    - 시간이 지날수록(또는 점수가 오를수록) 아이템이 떨어지는 속도가 빨라집니다.
    - 레벨업 시 "Level Up!" 문구 표시.
- **게임 오버 조건**:
    - 폭탄을 받았을 때 (Life 시스템인 경우 Life가 0이 되면 종료).
    - (옵션) 과일을 놓쳤을 때 (Life 감소).

## 6. UI 요소
- **상단 정보**: 현재 점수(Score), 레벨(Level), 남은 시간(Time) 또는 생명(Life).
- **중앙 화면**: 플레이어 캐릭터(바구니), 떨어지는 아이템들.
- **게임 종료 시**: 최종 스코어 팝업 및 '다시 시작' 버튼.
